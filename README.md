# Time Tracker — 本地时间记录工具

一个纯前端、零安装的时间记录单页应用，所有数据存储在浏览器 LocalStorage 中，无任何外部网络请求。

## 功能

- **月度日历视图** — 点击任意日期（过去/未来均可）进行补填或预填
- **项目与非项目分类** — 支持"项目"和"非项目活动"（出差、休假、培训、会议等）两大类，可自定义类别
- **加班预警** — 当日总时长超过 8 小时自动弹出警告，强制要求填写加班理由
- **月度总结报告** — 一键生成含统计图表的月度报告，小时自动转换为天（1 天 = 8 小时）
- **数据安全** — 所有数据仅存储在浏览器 LocalStorage，支持 JSON 导入/导出备份，CSV 导出
- **离线可用** — 无任何外部依赖，双击 HTML 文件即可使用

## 快速开始

1. 下载 `index.html`
2. 双击在浏览器中打开
3. 开始记录时间

无需 Node.js、无需安装、无需网络。

## 数据导出与备份

| 功能 | 说明 |
|------|------|
| 导出 JSON | 下载完整数据备份（含类别和记录） |
| 导入 JSON | 从备份恢复数据，自动合并类别 |
| 导出 CSV | 下载月度报告为 CSV，可在 Excel 中打开（含 BOM 头） |

## 时间换算规则

- 记录单位为**小时**，最小步长 0.25h（15 分钟）
- 月度报告中自动换算为**天**：`天数 = 小时数 ÷ 8`，保留两位小数
- 标准工时：8 小时/天

## 技术栈

纯 HTML/CSS/Vanilla JavaScript，单文件约 1200 行，无框架、无构建工具、无 npm 依赖。

# 业务数据自动化处理工具

模拟企业数据清洗、转换和报表生成的 ETL 流水线工具。

## 功能

- 支持 CSV / Excel 文件读取
- 自动去重、缺失值补全、异常值过滤
- 字段映射与格式标准化，自动计算派生字段
- 生成文本统计报表并导出处理后的 CSV

## 快速开始

```bash
pip install pandas openpyxl
python business_data_processor.py
```

首次运行会自动生成模拟数据并处理，输出 `processed_data.csv` 和 `processed_data.report.txt`。

## 处理真实数据

```python
from business_data_processor import run_pipeline

run_pipeline("你的业务数据.csv", output_dir="./output")
```

## 清洗规则

| 步骤 | 规则 |
|------|------|
| 去重 | 移除完全重复的行 |
| 缺失值 | 数值列用中位数填充，文本列填"未知" |
| 异常值 | 剔除数量 ≤0、单价超过 99 分位数、折扣率超出 [0, 0.8] 的记录 |

## 输出文件

- `processed_data.csv` — 清洗并转换后的数据
- `processed_data.report.txt` — 汇总报表（总订单数、金额指标、区域/品类 TOP 排名）
